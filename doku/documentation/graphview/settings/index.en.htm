Change graph view settings

The user has several possibilities to adjust the appearance of the graph view

  1. Adjust the number of entities shown in the graph (1). The user specified number of entities (e.g. 5) with highest frequency per type are shown).
  2. The user can show/hide edges according to their importance (2). Importance means edge frequency (i.e., co-occurrence of entities in documents).
  3. The user can adjust the view:
    1. Show/hide legend (4)
    2. Show graph in full screen mode. Resume full screen mode by pressing “ESC” button.
    3. Reset focus of the view.
    4. Zoom in or out in the view.
