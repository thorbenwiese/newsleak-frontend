Edit entities

The user can edit the network data. The user can edit the entities, merge entities or blacklist them.

Edit entity properties

The user is able to edit the extracted entities – he can edit their name and their type (person, organization, location, miscellaneous). Clicking on the “edit node” from the context menu opens an edit window. The changes are stored in the database and shown on the screen.

Merge nodes

The user can merge nodes. This is useful, if the user recognizes that two displayed nodes are actually the same.
This is a relatively easy task for the user, but not for the algorithm. For merging two nodes, the user needs to click on them with “CTRL” button clicked. And then click on “merge” from the context menu of one of the selected nodes. Then a merge window opens where the user can select which of the nodes will be the new “target” node and edit it. The changes are stored in the database and showed on the screen. The list of merged nodes can be seen and edited in settings.

Blacklist entities

The user can also blacklist nodes which are of no interest for him or which are wrongly extracted by the natural language algorithm. Blacklisted nodes are removed from the database and from the view. Blacklist can be selected from the node’s context menu. The list of blacklisted nodes can be seen and edited in settings.
